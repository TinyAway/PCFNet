# PCFNet
Official implementation of Pyramid correlation fusion Network
# Requirements
* Python 3.7
* Pytorch 1.9
* albumentations 0.4
* faiss 1.7
* tensorboardX 2.4
# Data preparation
Download the following datasets:
1.	[iSAID-5i](https://github.com/caoql98/SDM)
2.	[DLRSD](https://sites.google.com/view/zhouwx/dataset#h.p_hQS2jYeaFpV0)
